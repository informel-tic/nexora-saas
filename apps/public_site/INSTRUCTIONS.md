# public_site — Public Marketing Site

## Purpose

Static public-facing landing page for the Nexora platform. Presents the sovereign YunoHost supervision value proposition to potential clients.

## Entry Point

- Single `index.html` file — self-contained HTML + inline CSS.

## Conventions

- **Static HTML only** — no JavaScript, no build step.
- **French language** UI.
- **No authentication** required.
- **Served by**: nginx in production at the root domain.
